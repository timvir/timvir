# Grid

> CSS grid with a predefined template (columns, gaps etc)

```
<Grid>
  <div>ITEM</div>
  <div>ITEM</div>
  <div>ITEM</div>
  <div>ITEM</div>
  <div>ITEM</div>
</Grid>
```
