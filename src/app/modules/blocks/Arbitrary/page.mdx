The `<Arbitrary>` component is a composition of an input field, button, and `<Exhibit>`. The children passed to
this component are placed inside the `<Exhibit>`, and are provided with a custom React Context.

See the [component documentation](/docs/components/Arbitrary) for details about its usage.

This hook gives you access to the [React Context] created by the `<Arbitrary>` component. The context
contains a seed number which you can use to initialize a pseudorandom number generator (PRNG).

```javascript
const { seed } = useContext();
```
