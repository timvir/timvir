# Figma

The `<Figma>` component embeds a Figma file into the page. The element has a fixed aspect ratio (4:3).

<Exhibit>
  <Sample variant="basic" />
</Exhibit>

You can also link to specific nodes inside the files. For example to show a page, or a component.

<Exhibit>
  <Sample variant="basic" url="https://www.figma.com/file/JGrPhgJowS8w5IkcsaxKYo?node-id=71%3A28" />
</Exhibit>

## Usage

```jsx
<Figma url="https://www.figma.com/file/0FenYNPAo5wV7cmYPELaUE" />
```
