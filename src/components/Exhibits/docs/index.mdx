# Exhibits

The `<Exhibits>` component is a container for multiple [`<Exhibit>`](/docs/components/Exhibit) children.
It shows them all in a flex wrap container, and allows you to select one to focus on.

The primary use case is to show the same component in multiple variations, and once the user has
selected the one they are interested in then showing them additional details (eg. source code).

<Exhibits id="exhibits" codeId="code">
  <Exhibit source={`<source 1 />`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 80, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 2>\n\n</source>`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 90, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 3>\n</source>`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 95, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 4>\n\n\n\n</source>`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 100, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 5>\n\n\n</source>`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 120, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 6>\n</source>`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 140, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 7 />`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 160, borderRadius: 16 }} />
  </Exhibit>
  <Exhibit source={`<source 8>\n\n</source>`}>
    <div style={{ background: "magenta", opacity: 0.5, height: 32, width: 120, borderRadius: 16 }} />
  </Exhibit>
</Exhibits>
